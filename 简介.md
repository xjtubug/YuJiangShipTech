# YuJiang ShipTech — B2B 船用设备销售平台简介

## 平台概述

YuJiang ShipTech 是一个面向全球航运市场的专业 B2B 船用设备电子商务平台，为船厂、船东、船舶管理公司和海事供应商提供一站式的船用设备采购与销售解决方案。

---

## 核心功能特点

### 🌐 全球化多语言支持
- 支持 4 种语言：中文、英文、日文、阿拉伯文
- 阿拉伯文自动 RTL（从右到左）布局
- 自动翻译工具（中文→英文智能翻译）
- 面向全球船舶市场的国际化设计

### 🛒 专业B2B产品展示
- **多级分类体系**：支持父子分类，覆盖甲板机械、安全设备、导航系统等全品类
- **高级筛选系统**：按分类、价格区间、排序方式快速定位产品
- **产品对比功能**：支持多产品横向对比规格参数
- **多媒体展示**：产品多图、视频、PDF技术文档
- **结构化数据**：Schema.org 标记，提升搜索引擎展示效果

### 📋 完整询价-报价-订单流程
- **在线询价**：客户可在线提交询价，支持多产品询价购物车
- **智能报价**：管理员基于询价一键生成PDF报价单
- **报价审批**：多状态流转（草稿→审批→发送→接受/拒绝）
- **订单管理**：从报价到订单无缝转换
- **物流追踪**：订单状态实时更新，支持物流单号跟踪

### 📰 新闻资讯模块
- **公司新闻**：手动编辑发布公司动态与活动
- **行业新闻**：全球船舶行业热点资讯
- **图片轮播**：每条新闻支持多图轮播展示
- **分类管理**：行业新闻/公司新闻分类过滤
- **来源追溯**：行业新闻标注来源链接

### 📊 全面数据分析与报告
- **访客分析**：IP、国家、设备、来源、访问行为全方位追踪
- **产品分析**：浏览量、停留时长、询价次数深度分析
- **询价分析**：热门产品、国家分布、趋势图可视化
- **订单报表**：订单数据汇总与状态追踪
- **灵活导出**：支持PDF/CSV格式导出
- **定时报告**：自动按日/周/月/季/年发送报告到指定邮箱
- **多维时间筛选**：今日/本周/本月/本季/本年/自定义范围

### 👥 客户关系管理（CRM）
- **客户画像**：完整客户信息管理（公司、联系人、国家、标签）
- **线索评分**：自动评估客户价值（Cold/Warm/Hot/Very Hot）
- **跟进记录**：电话、邮件、会议全记录
- **历史追溯**：客户询价→报价→订单全链路历史

### 📧 邮件营销系统
- **模板管理**：多种邮件模板（欢迎、跟进、促销、节日）
- **营销活动**：手动/自动营销活动管理
- **发送追踪**：发送量、打开率、点击率统计
- **Newsletter**：网站订阅管理

### 🔍 SEO & 营销优化
- **结构化数据**：Organization、Product、BreadcrumbList Schema
- **Sitemap**：自动生成网站地图
- **元数据优化**：每页自定义 title/description/OG 标签
- **多语言 SEO**：hreflang 标签，多语言搜索引擎优化
- **UTM 追踪**：支持 UTM 参数来源追踪

### ⚡ 性能优化
- **懒加载**：所有页面非首屏组件延迟加载
- **图片优化**：Next.js Image 组件自动优化
- **路由预加载**：Next.js Link 自动预取
- **缓存策略**：revalidate 300s 服务端缓存
- **代码分割**：dynamic import 按需加载

### 🛡️ 安全特性
- **认证授权**：NextAuth.js 会话管理
- **密码加密**：bcrypt 加密存储
- **CSRF 防护**：内置 CSRF token
- **API 权限**：管理接口强制认证检查
- **SSL/HTTPS**：全站 HTTPS 加密
